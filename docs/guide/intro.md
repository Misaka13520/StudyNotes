# Markdown语法介绍

---

## Markdown 常用语法速查表

### 1. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

### 2. 文字样式

```markdown
**加粗文字**
*斜体文字*
***加粗+斜体***
~~删除线~~
<u>下划线</u>
`行内代码`
==高亮文字==（部分渲染器支持）
```

效果：**加粗** | *斜体* | ***加粗斜体*** | ~~删除线~~ | <u>下划线</u> | `行内代码`

---

### 3. 列表

#### 无序列表

```markdown
- 列表项 1
- 列表项 2
  - 嵌套项 2.1
  - 嵌套项 2.2
    - 更深嵌套
```

#### 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
   1. 嵌套有序 3.1
   2. 嵌套有序 3.2
```

#### 任务列表（待办事项）

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待做事项
```

---

### 4. 链接与图片

```markdown
[链接文字](https://example.com)
[带标题的链接](https://example.com "鼠标悬停提示")

![图片替代文字](图片地址)
![logo](/logo.png "可选标题")

<!-- 引用式链接 -->
[Google][1]
[1]: https://www.google.com
```

---

### 5. 引用

```markdown
> 这是一段引用
> 可以多行

> 嵌套引用
>> 第二层引用
>>> 第三层引用
```

> 引用效果展示：学习不是为了考试，而是为了能力的提升。

---

### 6. 代码

#### 行内代码

```markdown
使用 `console.log()` 来调试
```

#### 代码块（指定语言高亮）

````markdown
```javascript
function hello() {
  console.log('Hello, World!');
}
```

```python
def hello():
    print("Hello, World!")
```

```bash
echo "Hello, World!"
```
````

---

### 7. 表格

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |
```

效果：

| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |

---

### 8. 分隔线

```markdown
---
***
___
```

以上三种都会渲染为一条水平分隔线。

---

### 9. 转义字符

如果想显示 Markdown 中的特殊字符本身，在前面加 `\`：

```markdown
\*不是斜体\*
\# 不是标题
\[不是链接\]
```

---

### 10. 脚注

```markdown
这是一段正文，包含一个脚注[^1]。

[^1]: 这就是脚注的内容。
```

---

### 11. 数学公式（KaTeX / MathJax）

需要 VuePress 安装数学插件后才可使用。

```markdown
行内公式：$E = mc^2$

块级公式：
$$
\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n
$$
```

### 12. VuePress 特有容器（提示框）

VuePress 默认主题支持以下自定义容器：

```markdown
::: tip 提示
这是一个提示信息。
:::

::: warning 警告
这是一个警告信息。
:::

::: danger 危险
这是一个危险警告。
:::

::: details 点击展开详情
这里是折叠的详细内容，点击标题可以展开/收起。
:::
```

::: tip 提示
这是一个提示信息。
:::

::: warning 警告
这是一个警告信息。
:::

::: danger 危险
这是一个危险警告。
:::

::: details 点击展开详情
这里是折叠的详细内容，点击标题可以展开/收起。
:::

### 13. Emoji 表情

```markdown
:smile: :heart: :thumbsup: :rocket: :star:
```

---

### 14. HTML 标签（Markdown 中直接使用）

```markdown
<span style="color: red;">红色文字</span>
<kbd>Ctrl</kbd> + <kbd>C</kbd>
<mark>高亮标记</mark>
<br> <!-- 换行 -->
<center>居中文字</center>
```

效果：<kbd>Ctrl</kbd> + <kbd>C</kbd> 复制 | <mark>高亮标记</mark>

---

### 15. 目录生成

```markdown
[[toc]]
```

在 VuePress 中，`[[toc]]` 会自动生成当前页面的目录。
